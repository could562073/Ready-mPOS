# Architecture Decision Records (ADR)

架構決策紀錄 — 記錄重要技術決策的背景、選項與結果。

## Index

| ADR | Title | Status |
|-----|-------|--------|

## Format

每份 ADR 包含：
- **Context** — 為什麼需要做這個決定
- **Decision** — 我們決定做什麼
- **Alternatives Considered** — 考慮過哪些選項
- **Consequences** — 優缺點與注意事項
